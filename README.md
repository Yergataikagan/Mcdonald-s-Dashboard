# Mcdonald's Dashboard
Business Case Study
This Power BI dashboard was built to give McDonald's European operations a centralized view of restaurant performance across two main areas: financial health and country-level breakdown. 
  1. Financials Overview: https://github.com/Yergataikagan/Mcdonald-s-Dashboard/blob/main/Financials%20Overview.png
     The Financials page tracks key metrics like total revenue, gross profit, and quantity sold, and visualizes how revenue and profit trend month by month using an area chart, alongside a breakdown of sales by         payment method.
